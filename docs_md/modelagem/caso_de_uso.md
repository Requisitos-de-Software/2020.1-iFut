#  Casos de Uso
<div class="line"></div>

<p align="justify">&emsp;

O caso de uso é uma especificação do comportamento de uma funcionalidade do sistema.
<br>
Ele tem como finalidade descrever como será o uso de uma funcionalidade do sistema.
</p>
