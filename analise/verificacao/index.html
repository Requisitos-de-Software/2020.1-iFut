Definição

A Verificação é uma atividade, a qual envolve a análise de um sistema para certificar se este atende aos requisitos funcionais e não funcionais. Para a verificação dos artefatos do nosso projeto, será usada a técnica de inspeção.

O objetivo básico das inspeções é verificar se o modelo de requisitos está de acordo com a notação do modelo e com o que se espera desse modelo.
