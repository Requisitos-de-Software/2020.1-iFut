# UC18 - Verificar o  <a href="../../lexico/#extrato-do-jogo">extrato do jogo</a>

<p><b>Descrição:</b> A funcionalidade permite ver as informações dos jogos.</p>
<p><b>Ator principal:</b>  <a href="../../lexico/#usuario">usuário</a> geral</p>
<p><b>Pré-requisito:</b> Ver extrato dos jogos   </p>
<b>Fluxo Principal:</b>
<p class = "text-justify">1. O  <a href="../../lexico/#usuario">usuário</a> abre o aplicativo.</p>
<p class = "text-justify">2. Escolhe o <a href="../../lexico/#campeonato">campeonato</a> desejado</p>
<p class = "text-justify">3. Seleciona a rodada desejada</p>
<p class = "text-justify">4. Clica no jogo desejado</p>
<p class = "text-justify">5. Olha as informações que procura</p>
<b>Fluxo de exceção 01: </b> <a href="../../lexico/#usuario">usuário</a> sem autorização
<p class = "text-justify">1. O  <a href="../../lexico/#usuario">usuário</a> acessa o site</p>
<p class = "text-justify">2. O  <a href="../../lexico/#usuario">usuário</a> seleciona um jogo que ainda não ocorreu</p>
<p class = "text-justify">3. Aparece a  <a href="../../lexico/#mensagem">mensagem</a> de jogo não realizado</p>


<p class = "text-justify"><b>Pós-condição: </b>O  <a href="../../lexico/#usuario">usuário</a> verifica as informação das partidas. </p>
