Tornar-se Usuário HiFi


Título Tornar-se Usuário Hifi
Objetivo O Usuário pode adicionar se tornar um Usuário Premium e ter acesso à todas as funcionalidades do Deezer com a melhor qualidade disponibilizada.
Contexto Pré-condições: 1. Usuário possuir uma conta no Deezer.
2. Usuário estar logado.
3. Usuário ser um Usuário Free ou Usuário Premium.
Pós-condição: O Usuário passará a ser um Usuário HiFi e terá acesso à todas as funcionalidades do Deezer com a melhor qualidade disponibilizada.
Atores Usuário.
Sistema.
Recursos 1. Acesso à internet.
2. Possuir cartão de crédito.
3. Possuir conta no Deezer.
Episódios 1. O Usuário acessa o Deezer.
2. O Usuário acessa as configurações do aplicativo.
3. O Usuário seleciona o botão de "Gerencia minha conta".
4. O Usuário seleciona o botão "Assinar".
5. O Usuário seleciona o plano de Deezer HiFi.
6. O Usuário insere as informarções do seu cartão de crédito.
7. O Sistema disponibiliza 30 dias de Deezer HiFi sem taxas, e o Usuário se torna Usuário HiFi.
8. Após os 30 dias, o Sistema descontará a mensalidade do cartão de crédito disponibilizado pelo Usuário Hifi.
Restrições Cartão de crédito válido.
Exceção 1. A internet parar de funcionar no meio da ação.
2. A aplicação fechar no meio da ação.
3. O Usuário não forma de pagamento.
