Histórias de Usuário

  História de usuário é uma descrição concisa de uma necessidade do usuário do produto/requisito sob o ponto de vista desse usuário. A história de usuário busca descrever essa necessidade de uma forma simples e leve.
Para melhor organização, separamos as histórias em seus respectivos épicos, porém listamos todos eles logo abaixo.
