Épico: Gerenciamento de conteúdo

US39 Criar playlists

Eu, como usuário, desejo criar playlists para que eu possa adicionar conteúdos do meu gosto e ouvi-los em sequência.

Critérios de Aceitação:

  • Deve apresentar o botão para criar uma playlist.
  • Deve apresentar um campo para escrever o nome da playlist.
  • Deve apresentar botão para adicionar conteúdo na playlist.


US40 Curtir conteúdo

Eu, como usuário, desejo curtir um conteúdo específico para que eu possa acessá-los na página de Minha Música.

Critérios de Aceitação:

  • Deve apresentar o botão para curtir um conteúdo.
  • Deve adicionar músicas curtidas às Mais Queridas.
  • Deve adicionar álbuns curtidos aos álbuns favoritos.
  • Deve adicionar podcasts curtidos aos podcasts favoritos.
  • Deve adicionar playlists curtidas às playlists favoritos.
  • Deve adicionar artistas curtidos aos artistas favoritos.

US41 Baixar conteúdo

Eu, como usuário, desejo baixar um conteúdo específico para que eu possa acessá-los sem necessitar de conexão com a internet.

Critérios de Aceitação:

  • Deve ser usuário premium.
  • Deve apresentar botão para baixar conteúdo.
  • Deve adicionar conteúdo baixado nos Downloads.
  • Deve possibilitar a execução offline do conteúdo baixado.


US42 Compartilhar conteúdo

Eu, como usuário, desejo compartilhar um conteúdo específico para que eu possa postar letras ou conteúdos nas redes sociais.

Critérios de Aceitação:

  • Deve apresentar botão para compartilhar conteúdo.
  • Deve apresentar botão para escolher o local de compartilhamento.
  • Deve apresentar conexão com a conta escolhida.


US25 Fila de Espera

Eu, como usuário, desejo visualizar a fila de espera de conteúdo, para que eu possa saber qual conteúdo será tocado em seguida.

Critérios de Aceitação:

  • Deve apresentar um botão para acessar a fila de espera.
  • Deve apresentar os conteúdos que serão tocados na ordem.
  • Deve apresentar botão para excluir um conteúdo da fila de espera.
  • Deve ser possível selecionar um conteúdo da fila de espera para ser tocado imadiatamente.


US26 Adicionar conteúdo à Fila de Espera

Eu, como usuário, desejo adicionar um conteúdo específico na fila de espera, para que eu possa escutá-lo em seguida.

Critérios de Aceitação:

  • Deve apresentar um botão para adicionar conteúdo na fila de espera.
  • Deve apresentar uma mensagem de sucesso ao adicionar o conteúdo à fila de espera.
