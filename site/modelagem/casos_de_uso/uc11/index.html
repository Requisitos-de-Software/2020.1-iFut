Escutar um Conteúdo

Descrição

O usuário deve poder escutar conteúdos específicos.

Atores

Usuário
Sistema

Pré-condição

O usuário não deve estar escutando um conteúdo específico.

Fluxo de Eventos

Fluxo Principal

  1. O usuário entra no Deezer.
  2. O usuário acessa a página de Busca.
  3. O usuário faz a Busca de uma música.
  4. O sistema gera resultados em tempo real.
  5. O usuário escolhe o conteúdo desejado.
  6. O usuário agora pode escutar o conteúdo selecionado.


Fluxos Alternativos

FA01 )

  1. O usuário entra no Deezer.
  2. O usuário acessa a página de Minha Música.
  3. O usuário seleciona as Mais Queridas.
  4. O sistema mostra as músicas curtidas.
  5. O usuário escolhe a música desejada.
  6. O usuário agora pode escutar a música selecionada.

FA02 )

  1. O usuário entra no Deezer.
  2. O usuário acessa a página de Minha Música.
  3. O usuário seleciona as Playlists.
  4. O sistema mostra as playlists curtidas.
  5. O usuário escolhe a playlist desejada.
  6. O usuário agora pode escutar um conteúdo da playlist ou escutá-la de forma aleatória.

FA03 )

  1. O usuário entra no Deezer.
  2. O usuário acessa a página de Minha Música.
  3. O usuário seleciona os Álbuns.
  4. O sistema mostra os álbuns curtidos.
  5. O usuário escolhe o álbum desejado.
  6. O usuário agora pode escutar um conteúdo do álbum ou escutá-lo de forma aleatória.

FA04 )

  1. O usuário entra no Deezer.
  2. O usuário acessa a página de Minha Música.
  3. O usuário seleciona os Mixes.
  4. O sistema mostra os mixes curtidos.
  5. O usuário escolhe o Mix desejado.
  6. O usuário agora pode escutar um conteúdo do álbum ou escutá-lo de forma aleatória.

FA05 )

  1. O usuário entra no Deezer.
  2. O usuário acessa a página de Minha Música.
  3. O usuário seleciona os Podcasts.
  4. O sistema mostra os podcasts curtidos.
  5. O usuário escolhe o podcast desejado.
  6. O usuário agora pode escutar um conteúdo do álbum ou escutá-lo de forma aleatória.

FA06 )

  1. O usuário entra no Deezer.
  2. O usuário acessa a página de Início.
  3. O usuário seleciona um conteúdo da página de Início.
  4. O usuário agora pode escutar o conteúdo selecionado.

Fluxos de Exceção

FE01) Conteúdo não encontrado

  1. O usuário entra no Deezer.
  2. O usuário acessa a página de Busca.
  3. O usuário faz a Busca de uma música.
  4. O sistema não encontra o conteúdo buscado.
  5. O usuário deve buscar por outro conteúdo.

FE02) Conteúdo não disponível

  1. O usuário entra no Deezer.
  2. O usuário acessa a página de Minha Música.
  3. O usuário seleciona os Álbuns.
  4. O sistema mostra os álbuns curtidos.
  5. O usuário escolhe o álbum desejado.
  6. O usuário seleciona um conteúdo do álbum que não está disponível.
  7. O sistema não executará o conteúdo selecionado e tocará o próximo conteúdo do álbum.

Pós-condição

O usuário agora pode escutar um conteúdo desejado.
